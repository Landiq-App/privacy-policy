8. Location Data and Permissions

8.1 How We Use Location Data

LandIQ uses location data in a LIMITED and SPECIFIC manner:

When Location is Used:

  • ONLY when you tap the "Use Current Location" button
  • NOT tracked continuously or in the background
  • NOT monitored without your knowledge
  • NOT stored separately from analyses

What Location Data We Collect:

  • Approximate GPS coordinates (latitude and longitude)
  • Accuracy level of coordinates
  • Timestamp of when coordinates were captured

How Location is Used:

  • To auto-fill property address field
  • To include coordinates in analysis report
  • To enable location-based property analysis

Where Location is Stored:

  • Coordinates included in analysis report (if generated)
  • Stored with analysis history (Pro users only)
  • NOT stored in separate location history database

8.2 Android Location Permissions

Permission Types:

  • Foreground Location: Required to use "Use Current Location" feature
  • Background Location: NOT requested or used by LandIQ

How to Control:

  • Grant permission: Respond to Android's prompt
  • Revoke permission: Settings > Apps > LandIQ > Permissions > Location > Deny
  • Change permission: Modify anytime in device settings

8.3 Location Privacy Protections

We protect your location privacy by:

  • Never tracking location in background
  • Never creating location history profiles
  • Never sharing raw coordinates with third parties (except as part of analysis to Claude AI)
  • Never using location for advertising or marketing
  • Never correlating location with other users

Location Alternatives: You can ALWAYS enter addresses manually with no disadvantage to analysis quality.
